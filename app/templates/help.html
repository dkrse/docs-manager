Help Manual

Document Manager is a web application for storing and organizing documents. Supported file types:

  • PDF — with thumbnail preview and in-browser viewer
  • DOCX — Microsoft Word documents, rendered as HTML
  • MD — Markdown files, rendered with formatting
  • TXT — plain text files

Each document can have a category, hashtags, description, notes, document date, and private/public visibility.

  1. Click Upload in the navigation bar.
  2. Select a file (PDF, DOCX, MD, or TXT).
  3. For PDF files, you can choose which page to use as the thumbnail preview.
  4. Fill in optional metadata:
    • Category — e.g. Invoice, Report, Contract
    • Hashtags — comma-separated tags for organizing, e.g. finance, 2024, quarterly
    • Description — short summary of the document
    • Notes — additional notes for your reference
    • Document Date — the date the document relates to (not the upload date)
    • Private — check this to make the document visible only to you and admins
  5. Click Upload.
Duplicate files are automatically detected. If a file with the same content already exists, the upload will be rejected.

The dashboard shows all your documents as tiles. Each tile displays the filename, upload date, and file size.

Actions on each tile:
  • Click the tile — opens the document viewer (PDF in iframe, others rendered as HTML)
  • Edit — edit the document metadata (category, tags, description, notes, date, visibility)
  • Download — download the original file
  • Delete — permanently delete the document (with confirmation)

Full-text search

Type in the Search field to search across:

  • Filename
  • Description
  • Notes
  • Hashtags
  • Document content — the actual text inside PDF, DOCX, MD, and TXT files

Search works without diacritics — e.g. searching for resume will find documents containing "Résumé".

Results update automatically as you type.

Filters

Click the Filter button to show advanced filters:

  • Category — filter by document category
  • Type — filter by file type (PDF, DOCX, MD, TXT)
  • Sort — change sort order (random, upload date, document date, file size)
  • Doc date — filter by document date range
  • Upload date — filter by upload date range

Click the icon in the navigation bar to access settings:

  • Theme — switch between Light and Dark mode
  • Documents per page — number of tiles shown per page (5–100)
  • Default sort — the default sort order when you open the dashboard
  • Hidden hashtags — comma-separated list of tags; documents with these hashtags will be hidden from your dashboard

  • Public documents — visible to all users
  • Private documents — visible only to the uploader and admins

You can change visibility at any time by editing the document metadata.

Admin users have additional capabilities:

  • View all documents (including private documents from all users)
  • Edit and delete any document
  • Manage users via Users page — create, edit, and delete user accounts
Admin users cannot upload documents. Only regular users can upload.

Click your username in the navigation bar to access the password change form. You need to enter your current password and the new password twice.
