Help Manual

Document Manager is a web application for storing and organizing documents. Supported file types:

  • PDF — with thumbnail preview and in-browser viewer
  • DOCX — Microsoft Word documents, rendered as HTML
  • XLSX — Excel spreadsheets, rendered as HTML tables
  • CSV — comma-separated values, rendered as HTML table
  • MD — Markdown files, rendered with formatting, Mermaid diagrams and LaTeX formulas (KaTeX)
  • TXT — plain text files

Each document can have a category, hashtags, description, notes, document date, and private/public visibility.

  1. Click Upload in the navigation bar or toolbar — opens a modal dialog.
  2. Select one or multiple files (PDF, DOCX, XLSX, CSV, MD, or TXT).
  3. For PDF files, you can choose which page to use as the thumbnail preview.
  4. Fill in optional metadata (applies to all selected files):
    • Category — e.g. Invoice, Report, Contract
    • Hashtags — comma-separated tags, e.g. finance, 2024, quarterly
    • Description — short summary of the document
    • Notes — additional notes for your reference
    • Document Date — the date the document relates to
    • Private — make the document visible only to you and admins
  5. Click Upload. A progress bar shows the status for each file.
Duplicate files are automatically detected. If a file with the same content already exists, that file will be skipped and the rest will continue uploading.

The dashboard has two display modes, toggled with the / button:

Grid View
  • Documents shown as tiles with thumbnails (PDF) or file type icons
  • Filename displayed below the tile
  • Action buttons (edit, download, delete) inside the tile
  • Star icon to add/remove from favorites
  • Checkbox for bulk selection
List View
  • Documents shown in a table with columns: Name, Type, Size, Created, Modified
  • Click any column header to sort — click again to reverse. Sorting works across all pages.
  • "Select all" checkbox in the header row
  • Action buttons in the last column
Actions on each document:
  • Click the name/thumbnail — opens the document viewer modal
  • Favorite — star/unstar the document
  • Edit — edit metadata (category, tags, description, notes, date, visibility)
  • Download — download the original file
  • Delete — permanently delete the document

Clicking a document opens a modal with the document content and metadata.

Markdown features

Markdown files support extended rendering:

  • Mermaid diagrams — fenced code blocks with language mermaid are rendered as diagrams (flowcharts, sequence diagrams, etc.)
  • LaTeX formulas — inline formulas with $...$ and display formulas with $$...$$ are rendered via KaTeX
  • Tables, fenced code blocks, and code highlighting
Actions

Available actions in the modal footer:

  • Share — generate a public link (see Sharing section)
  • Open — open the document in a new tab (full page). Uses secure daily-rotating URLs.
  • Download — download the original file

Metadata search

Type in the Search field to search across metadata:

  • Filename
  • Description, Notes, Hashtags

Search is diacritics-insensitive — e.g. searching for resume will find "Résumé". Results update automatically as you type.

Search in files

Click the in files button to open a dedicated search modal for searching inside document content (PDF, DOCX, XLSX, CSV, MD, TXT):

  • Enter a search term and click Search (or press Enter)
  • Real-time progress shows the name of each file being searched
  • Results appear incrementally as matches are found, with the matching line highlighted
  • Click any result to open that document
  • After search completes, the main document list is automatically filtered to show only matched documents
  • All active filters (category, type, dates, size, favorites) are respected
Filters

Click the Filter button to show advanced filters:

  • Category — filter by document category
  • Type — filter by file type (PDF, DOCX, XLSX, CSV, MD, TXT)
  • Sort — change sort order (random, upload date, document date, file size)
  • Favorites — show only starred documents
  • Size (KB) — filter by file size range (from/to in kilobytes)
  • Doc date / Upload date — filter by date range

Select multiple documents using the checkboxes on tiles (grid) or rows (list). A bulk action bar appears with:

  • ZIP Download — download all selected documents as a single ZIP file
  • Delete — delete all selected documents (with confirmation)
  • Clear — deselect all

In list view, use the checkbox in the header to select/deselect all documents on the current page.

Click the star icon on any document to mark it as a favorite. Favorites are per-user.

  • In grid view, the star is in the top-right corner of the tile
  • In list view, the star is in the second column
  • Use the Favorites filter to show only starred documents

You can share a document publicly without requiring the recipient to log in:

  1. Open the document in the viewer modal
  2. Click the Share button
  3. A public link is generated. Click to copy it to clipboard.
  4. Send the link to anyone — they can view and download the document without an account.

To revoke sharing, click the button next to the link. The link will stop working immediately.

Click the icon in the navigation bar to open the settings modal:

  • Theme — switch between Light and Dark mode
  • Documents per page — number of documents shown per page (5–100)
  • Default sort — the default sort order when you open the dashboard
  • Hidden hashtags — comma-separated list of tags; documents with these hashtags will be hidden
Tile controls visibility

Toggle which action buttons are shown on document tiles:

  • Edit — show/hide the edit button
  • Download — show/hide the download button
  • Delete — show/hide the delete button

  • Public documents — visible to all logged-in users
  • Private documents — visible only to the uploader and admins
  • Shared documents — accessible to anyone with the share link (no login needed)

You can change visibility at any time by editing the document metadata.

Admin users have additional capabilities:

  • View all documents (including private documents from all users)
  • Edit and delete any document
  • Manage users via Users page — create, edit, and delete user accounts
Admin users cannot upload documents. Only regular users can upload.

Click your username in the navigation bar to open the password change modal. Enter your current password and the new password twice.
